========================================
HOSPITALITY MODE – CALCULATIONS
========================================

Derived values:
- NightsInSeason_s = number of nights between StartDate_s and EndDate_s
  Decide inclusive/exclusive:
  Recommended: Nights = difference in days (End - Start), where End is checkout day (exclusive).
  If you want inclusive end date, then Nights = (End - Start) + 1.
  Choose one and label it in UI: “End date is last night” vs “End date is checkout date”.

Per season s, per room type r:
- AvailableRoomNights_(r,s) = Rooms_r × NightsInSeason_s
- OccupiedRoomNights_(r,s) = AvailableRoomNights_(r,s) × (Occupancy%_s / 100)
- Revenue_(r,s) = OccupiedRoomNights_(r,s) × RatePerNight_(r,s)

Season totals:
- SeasonRevenue_s = Σ_r Revenue_(r,s)

Annual totals:
- AnnualRoomRevenue = Σ_s SeasonRevenue_s
- EGI_used = AnnualRoomRevenue + OtherAnnualIncome

NOI:
- NOI = EGI_used − OperatingExpenses + UtilityAdj

Value Range:
- capLow = CapLow% / 100
- capHigh = CapHigh% / 100
- ValueHigh = (NOI ÷ capLow) + ExcessLand − Refurb
- ValueLow  = (NOI ÷ capHigh) + ExcessLand − Refurb
Display: Value Range = min(ValueLow, ValueHigh) to max(...)

Useful output metrics:
- Total Rooms = Σ Rooms_r
- Total GLA (m²) = Σ(Rooms_r × Size_m²_r)
- Weighted Average Occupancy:
  WeightedOcc = Σ_s Σ_r OccupiedRoomNights_(r,s) ÷ Σ_s Σ_r AvailableRoomNights_(r,s)
- Weighted ADR:
  ADR = AnnualRoomRevenue ÷ Σ_s Σ_r OccupiedRoomNights_(r,s)
- Revenue per season table + grand total
